# gitui

# keybind

in vim-key-bind.

- ? : help
- q : quit
- ESC : back
- TAB : move the tab.
- number : move the tab.
- k,j : move up/down
- l : next.

## Status [1]

- w : move the "Unstaged Changes" window <-> the "Staged Changes" window
- ENTER: staging / unstaging the file
- s : staging / unstaging at line.
- c : commit.
