# tauri

[tauri studio](https://tauri.studio/en/)

```sh
$ npx create-tauri-app $APP_NAME
```

## install
```sh
$ npm init
$ npm install @tauri-apps/cli
$ cat package.json
{
  "script": {
    "tauri": "tauri"
  }
}
$ npm run tauri init
```
